Introduction


NSX-T 3.0

  • full stack Layer 2 to Layer 7 networking platform that offers virtual networking, security, load balancing, visibility, and analytics in a single platform
  • With NSX-T 3.0, admins can now deploy NSX-T directly on VMware vSphere Distributed Switch 7.0
  • Versioning
    • NSX-V
      • Require a NSX manager instance for every vcenter (maintenance can be complicated)
      • No GUI, just a plugin integrated in vcenter
    • NSX-T
      • Implement agile infrastructure
      • Many vcenter can be register to 1 NSX manager
      • Dedicated GUI
      • It works without vcenter


NSX Portfolio

  • NSX Data Center
  • NSX Cloud extends the data center network into the public cloud (VMware Cloud on AWS or Microsoft Azure Cloud)
  • NSX Intelligence is a distributed analytics solution that provides visibility and dynamic security policy enforcement for NSX-T Data Center environments
  • NSX Distributed IDS/IPS
  • NSX Advanced Load Balancer
  • NSX Service Mesh enables discovery and secure communication of microservices in a heterogeneous infrastructure
  • VMware HCX delivers application and network hybridity and mobility


Virtual Cloud Network

  • VMware Framework
  • Software layer


NSX-T Data Center

  • It’s part of Virtual Cloud Network
  • Security, Multi-cloud networking, automation, Cloud-Native Application
